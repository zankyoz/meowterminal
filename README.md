# meowterminal ฅ^•ﻌ•^ฅ

a pink terminal that lives in your browser ♡  
a browser-based terminal simulator with a virtual filesystem, command parser, and persistent storage.  
type commands, make notes, run javascript.

---

## what is this (ﾉ◕ヮ◕)ﾉ

it's a fully functional fake terminal but make it cute!!  
pink. glowy. has pixel hearts floating around.  
built with plain html, css & js — no dependencies, no frameworks.

---

## features ✦

- ✦ scanline effects + floating pixel hearts
- ✦ a fake filesystem u can actually navigate (`ls`, `cd`, `mkdir`, `rm`, etc.)
- ✦ persistent notes + key-value storage (saves to localStorage!)
- ✦ run real javascript in the terminal with `js <code>`
- ✦ built-in calculator with `calc <expr>`
- ✦ kaomoji commands: `hello`, `uwu`, `meow`, `love`, `stats`
- ✦ command history with arrow keys

---

## how to use (˶ᵔ ᵕ ᵔ˶)

just open the html file in any browser!! that's literally it~

```
open index.html
```

no install. no build step. no node_modules trauma. ♡

## commands ✿

| command | what it does |
|---|---|
| `help` | shows all commands |
| `ls` | list files in current dir |
| `cd <path>` | change directory |
| `cat <file>` | read a file |
| `touch <file>` | create a file |
| `mkdir <dir>` | make a new directory |
| `rm <name>` | delete a file or dir |
| `mv <src> <dst>` | move / rename |
| `cp <src> <dst>` | copy a file |
| `write <file> <text>` | write text to a file |
| `js <code>` | run real javascript!! |
| `calc <expr>` | math calculator |
| `note <text>` | save a note |
| `notes` | see all your notes |
| `set <key> <val>` | store a value |
| `get <key>` | retrieve a value |
| `keys` | list stored keys |
| `del <key>` | delete a key |
| `clearstorage` | wipe all saved data |
| `echo <text>` | print text |
| `whoami` | who r u |
| `date` | current date & time |
| `history` | command history |
| `clear` | clear the screen |
| `hello` | get a warm greeting (˶˃ᵕ˂˶) |
| `uwu` | uwu |
| `meow` | meow |
| `love` | ♡ ♡ ♡ |
| `stats` | check ur stats |
